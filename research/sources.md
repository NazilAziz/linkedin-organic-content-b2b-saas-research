## Core Deliverables

### LinkedIn posts

Two annotated posts per expert in [`linkedin-posts/`](linkedin-posts/). Each file includes post date, original URL, and summary.

### Expert profiles

One profile per expert in [`other/experts/`](other/experts/). Each file includes profile links, source URLs, selection rationale, and priority sources.

### Other reference material

Frameworks, newsletters, reports, and additional notes go in [`other/`](other/).
